DOM XSS Site

by Paul "P-cap" Capili

Synopsis

According to OWASP, DOM-based is an XSS attack wherein the attack payload is executed as a result of modifying the DOM “environment” in the victim’s browser used by the original client side script, so that the client side code runs in an “unexpected” manner.
